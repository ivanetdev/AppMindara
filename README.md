# AppMindara
Software para una clinica de psicología, la cual necesita insertar/editar usuarios, terapeutas y pacientes, así como informes, citas o tratamientos. 
Con 2 versiones: 
  1- Con BBDD persistente y necesidad de XAMPP y PHPMyAdmin.
  2- Sin BBDD pesistente y sin instalar nada (datos no se guardan)
